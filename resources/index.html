Hacking the Meta Quest for FRC Navigation

2025 Jumpstart
Description
Learn how to use a Meta Quest VR headset to give your robot spatial awareness. We will dive into QuestNav — a system that uses the Quest's advanced capabilities to track your robot's position on the field with centimeter precision, even without AprilTags. This session covers the hardware setup and the Java code required to integrate it with your robot.
Presented by
Andy Gasser
Team 7028 Mentor • Control Systems Advisor (CSA) • STMA Robotics board member • FTC Mentor
